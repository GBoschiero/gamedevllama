Updated Top-Down Token Rendering Pipeline

A refined method for rendering clean, readable top-down tokens from 3D characters.

1. Camera Setup

I switched to a pure orthographic camera to ensure consistency between tokens and maps.

2. Lighting Pass

Soft rim lighting improves silhouette readability without flattening the model.

3. Rendering Process

  • Batch import poses
  • Auto-rotate actor to match token direction
  • Render via high-res render queue
  • Autocrop → export → pack

4. Result

The new tokens match Foundry & DungeonDraft scale perfectly.
