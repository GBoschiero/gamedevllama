Llama VTT – First Look at the Grid System

Breaking down the very first version of the VTT grid engine.

1. Goals of the Grid Prototype

The first goal was not beauty — it was functionality. The grid needed to support:

  • Snapping
  • Selection + highlight
  • Tile coordinate translation
  • Future support for fog of war & lighting

2. World-Space Cell Structure

The system uses world space, not screen space. This gives future compatibility with 3D terrain, height, ramps, and 3D dice.

3. Interaction Layer

Interaction is handled by a raycast grid mapper, not by colliders, which keeps performance high.

Next Steps

The next iteration will include grid masking, terrain elevation, and biome blending.
