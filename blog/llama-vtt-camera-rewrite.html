Rewriting the VTT Camera System

Improving navigation smoothness, constraints, and cinematic potential.

1. Why Rewrite It?

The original camera was functional but stiff. It didn’t support angles I want for dice animations, or smooth map exploration.

2. New Features

  • Smooth pan using friction curves
  • Zoom sensitivity based on map scale
  • Orbiting camera for cinematic moments
  • Map-edge constraints

3. Result

The VTT feels significantly more “alive” to navigate.
