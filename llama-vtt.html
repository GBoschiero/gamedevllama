Llama VTT

The fully custom Virtual Tabletop built in Unreal Engine.

Llama VTT is my long-term project — a virtual tabletop built from scratch featuring:

  • Custom grid and map systems
  • Procedural environment tools
  • Top-down token rendering pipeline
  • Modular actor system
  • Cinematic features
  • Highly visual, Unreal-powered UI
